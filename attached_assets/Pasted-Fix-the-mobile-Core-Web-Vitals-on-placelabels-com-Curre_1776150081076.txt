FIX 1 — Eliminate render-blocking Google Fonts (saves 910ms on mobile)

The current font loading in index.html uses preload + onload swap,
but it's still blocking on mobile. Replace it entirely with this 
optimized approach:

REMOVE these lines:
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link rel="preload" as="style" href="https://fonts.googleapis.com/css2?
family=Inter:wght@400;500;600;700;800&display=swap" 
onload="this.onload=null;this.rel='stylesheet'">
<noscript><link rel="stylesheet" href="..."></noscript>

REPLACE WITH this single non-blocking tag:
<link rel="stylesheet" 
href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;
700;800&display=swap&display=swap" 
media="print" onload="this.media='all'">
<noscript>
<link rel="stylesheet" 
href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;
700;800&display=swap">
</noscript>

Then add this to the main CSS file as a fallback so text shows 
immediately without waiting for Inter to load:
body { font-family: Inter, -apple-system, BlinkMacSystemFont, 
'Segoe UI', sans-serif; }

FIX 2 — Move Google Analytics to load after page paint (saves ~200ms)

In index.html, move the GA script block to load after the page 
is interactive. Replace the current inline GA block:

REMOVE:
<script async src="https://www.googletagmanager.com/gtag/js?
id=G-3DYJNWSKJP"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'G-3DYJNWSKJP');
</script>

REPLACE WITH (deferred load after page interactive):
<script>
  window.addEventListener('load', function() {
    var s = document.createElement('script');
    s.src = 'https://www.googletagmanager.com/gtag/js?id=G-3DYJNWSKJP';
    s.async = true;
    document.head.appendChild(s);
    window.dataLayer = window.dataLayer || [];
    function gtag(){dataLayer.push(arguments);}
    window.gtag = gtag;
    gtag('js', new Date());
    gtag('config', 'G-3DYJNWSKJP');
  });
</script>
