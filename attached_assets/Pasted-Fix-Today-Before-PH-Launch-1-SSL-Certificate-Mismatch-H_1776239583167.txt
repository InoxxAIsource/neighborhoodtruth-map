🟠 Fix This Week — Replit Prompt
Add Gzip/Brotli compression and cache headers to the Express server.

STEP 1 — Install compression:
npm install compression
npm install --save-dev @types/compression

STEP 2 — Add to Express server (BEFORE static middleware):
import compression from 'compression';
app.use(compression());

STEP 3 — Add cache headers for static assets:
app.use(express.static(path.join(__dirname, 'dist'), {
  maxAge: '1y',
  etag: true,
  setHeaders: (res, path) => {
    if (path.endsWith('.html')) {
      // Don't cache HTML — always fresh
      res.setHeader('Cache-Control', 'no-cache');
    }
  }
}));

STEP 4 — Add HSTS header:
app.use((req, res, next) => {
  res.setHeader('Strict-Transport-Security', 
    'max-age=31536000; includeSubDomains');
  next();
});

After deploying, verify:
curl -I https://placelabels.com/assets/index.js
Should show: Content-Encoding: gzip
             Cache-Control: max-age=31536000
