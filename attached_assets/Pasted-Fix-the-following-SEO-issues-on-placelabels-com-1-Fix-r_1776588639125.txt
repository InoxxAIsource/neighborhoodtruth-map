Fix the following SEO issues on placelabels.com:

**1. Fix robots.txt and sitemap.xml routing (CRITICAL)**
The app's SPA router is catching /robots.txt and /sitemap.xml and returning a "City Not Found" page. Fix this by:
- Serving a static robots.txt file at the root with:
  User-agent: *
  Allow: /
  Sitemap: https://placelabels.com/sitemap.xml
- Creating a /sitemap.xml route or static file that lists https://placelabels.com/ and any other public pages
- Make sure these paths are excluded from the SPA city-slug routing logic (add them as exceptions before the catch-all route)

Change the current H1 (which just says "PlaceLabels") to something descriptive like:
